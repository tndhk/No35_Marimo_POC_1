# Initial Concept
A monorepo-based platform for interactive data analysis using marimo notebooks, focusing on modular analysis projects like Bento sales prediction and Titanic survival prediction.

# Product Guide

## ビジョン
データ分析の学習者が、インタラクティブな環境を通じて実践的なスキルを習得できるプラットフォーム。Pythonの基礎から機械学習のライフサイクル、そして可視化までをシームレスに体験できる場を提供します。

## ターゲットユーザー
- **学習者・学生**: データ分析のパターンを学び、手を動かしながら理解を深めたいと考えている初心者。

## 主な機能・特徴
- **新規分析プロジェクトの拡充**: 不動産価格予測やワイン品質評価などのオープンデータセットを用いた、新しい学習用プロジェクトを順次追加。
- **教育的サポート**: 概念を解説する包括的なMarkdownチュートリアルや、コード内の詳細なインラインコメントを通じて、背景にある理論と実装を同時に学習可能。
- **実践的アウトプット**: marimoの特性を活かし、パラメータ変更が即座に反映されるインタラクティブで共有可能なデータレポートの作成手法を学習。

## 学習成果
- **データ操作の基礎**: PythonおよびPolarsを用いた高速なデータ処理スキルの習得。
- **MLライフサイクルの理解**: データの前処理、特徴量エンジニアリング、モデル構築、評価までの一連の流れを包括的に理解。
- **インタラクティブな可視化**: Altairやmarimoウィジェットを用いた、効果的なデータストーリーテリングの習得。
